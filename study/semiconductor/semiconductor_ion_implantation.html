이온주입 공정 (Ion Implantation)

1. 개요

  • 반도체가 전기적인 특성을 갖도록 만드는 과정
  • 전기가 통하는 도체와, 통하지 않는 부도체의 성질을 동시에 가진 반도체
  • 순수한 반도체는 규소로 되어있어 전기가 통하지 않으나 불순물을 넣어 전류가 흐르게 하는 전도성을 갖게 되는 것
  • 이때 불순물을 이온(Ion)이라고 함
    • 15족 원소 인(P), 비소(As), 13족 원소 붕소(B)등을 사용
    N형 반도체 = 15족 원소를 주입한 반도체
    P형 반도체 = 13족 원소를 주입한 반도체
  • 도핑(Doping) = 전기가 잘 흐르게 하거나, 특정 방향으로만 흐르게 만드는 과정 = 이온주입 공정

2. 공정의 흐름

＊ 원자의 종류

  • N형 도핑 : 전자 증가, 대표적으로 Phosphorus (P), Arsenic (As)
  • P형 도핑 : 정공 증가, 대표적으로 Boron(B)
  • 핵심 원리
    • "이온을 전기적으로 가속해서 실리콘에 충돌시킴"

① 이온 생성 (Ion Source)

  • 목적 : 도핑 원자를 이온화
  • 고온의 플라즈마를 사용하여 전자 충돌로 이온을 생성
  • 예시
    • BF3 → B+
    PH3 → P+
    AsH3 → As+

② 이온 추출 (Extraction)

  • 생성된 이온을 전압을 인가하여 전기장으로 끌어냄 → Ion beam 생성

③ Mass Analyzer ★★★

  • Ion source에는 원하는 이온 + 불순물이 다 섞여 있음
    • B+, BF2+, 기타 이온
  • 자기장을 사용하여 질량마다 휘는 정도가 다른 이온의 특성을 이용하여 원하는 이온을 걸러낸다
  • 로렌츠 힘 : F = q(v X B)
  • 질량이 작은 것은 많이 휘고, 질량이 큰 것은 상대적으로 덜 휨

④ Acceleration ★★★

  • 실리콘 내부에 침투시키기 위해 이온을 고속으로 가속 시킴
  • 에너지 = 침투깊이, 에너지가 높을수록 더 깊이 들어감 (에너지의 범위 : 수 keV ~ MeV)

⑤ Implantation

  • 가속 시킨 이온을 웨이퍼에 충돌시켜 주입 시키는 과정
  • 이온이 Si(규소) 내부 원자와 충돌하면서 에너지를 잃고 특정 깊이에서 멈추게 됨
  • Projected Range (Rp) = 평균 침투 깊이
  • 분포 형태는 완벽히 일정하지 않고, 가우시안 분포와 비슷한 형태
