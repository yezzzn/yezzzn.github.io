소프트웨어 (Software)

1. 개요

  • 머신 비전 시스템에서 이미징 소프트웨어는 하드웨어와 응용 프로그램 사이를 연결하는 핵심 역할을 담당
  • 이미징 소프트웨어의 주요 역할
    • 카메라 하드웨어와의 인터페이스
    카메라 기능 제어 (노출, 게인, 트리거 등)
    이미지 수신, 처리 및 분석
    이미지 결과를 이용한 의사결정 수행
    다른 연결된 장치 또는 기계와 통신

2. 소프트웨어 종류

① 카메라 뷰어 (Camera Viewer)

  • 카메라와 네트워크 환경을 간단하게 설정하고 이미지를 실시간으로 확인할 수 있는 기본 소프트웨어
  • 주요 기능
    • 네트워크 구성 : 카메라를 네트워크에 연결하고 IP 설정
    카메라 구성 : 노출, 게인, 해상도 등 카메라 파라미터 설정
    이미지 캡처 / 향상 / 표시 / 기록
    분석 도구 : 히스토그램, 라인 플롯(Line Plot) 등 이미지 분석 기능 제공

② 포괄적인 소프트웨어 (Comprehensive Software)

  • 카메라 뷰어의 모든 기능을 포함하며, 복잡한 멀티카메라 환경과 고급 이미지 처리를 위한 확장 기능을 제공
  • 주요 기능
    • 프로젝트 관리 : 검사 프로젝트 단위로 설정 저장 및 관리
    다중 카메라 / 서버 구성 : 여러 대의 카메라를 동시에 제어하고 구성
    GPU 지원 : GPU 가속을 활용한 고속 이미지 처리
    3D 이미지 재구성 : 2D 이미지 데이터를 기반으로 3D 형상 생성
    3D 개체 뷰어 : 재구성된 3D 데이터를 시각적으로 탐색

③ SDK (Software Development Kit)

  • 개발자가 자체 애플리케이션에 카메라 기능을 통합할 수 있도록 제공되는 개발 도구 모음
  • 주요 구성 요소
    • 코드 샘플 : 주요 기능 구현을 위한 예제 코드 제공
    라이브러리 : 카메라 제어 및 이미지 처리를 위한 함수 라이브러리
    API : 카메라와 소프트웨어 간 인터페이스 명세
    문서 : 상세 사용 설명서 및 레퍼런스 매뉴얼
    컴파일러 / 디버거 : 개발 및 디버깅 환경 지원
  • 지원 언어 : C / C++ / C# / Python
  • 지원 플랫폼 : Windows / Linux / ARM
