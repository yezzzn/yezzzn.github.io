인터페이스 주변기기 (Interface)

1. 개요

  • 인터페이스 주변기기는 카메라와 컴퓨팅 플랫폼 사이에서 이미지 데이터를 전달하는 역할을 담당
  • 사용하는 케이블 규격(Ethernet, USB, CoaXPress, Camera Link)에 따라 필요한 주변기기가 달라짐
  • 인터페이스 주변기기의 선택은 전송 속도, 케이블 길이, 다중 카메라 구성 여부 등을 고려하여 결정

2. Ethernet 인터페이스

① 네트워크 인터페이스 카드 (NIC, Network Interface Card)

  • PCIe 슬롯을 통해 컴퓨터 마더보드에 연결하는 확장 카드
  • OSI 모델의 물리 계층 및 데이터 링크 계층에 대한 전자 회로를 구현
  • 고속 GigE 카메라 연결에 필수적이며, 마더보드 내장 이더넷 포트 대비 높은 버퍼 크기와 성능을 제공

② 스위치 (Switch)

  • 동일한 LAN(Local Area Network) 내에서 이더넷 장치를 연결하는 장비
  • 수신한 패킷의 헤더를 읽어 대상 주소를 파악하고 해당 포트로만 전달 (브로드캐스트 없음)
  • 머신 비전에서 스위치가 필요한 경우
    • 멀티플렉싱 : 고속 멀티 카메라 시스템에서 여러 이더넷 포트의 픽셀 데이터를 통합하여 단일 포트로 출력
    멀티캐스팅 : 카메라 데이터를 여러 호스트 PC로 전송
    PTP(Precision Time Protocol) : 여러 카메라의 타임스탬프를 단일 장치 시계와 동기화

③ 이더넷 허브 (Ethernet Hub)

  • 여러 장치를 연결할 수 있으나, 스위치와 달리 연결된 모든 장치로 패킷을 브로드캐스트
  • 스위치 대비 효율이 낮아 현재 머신 비전 환경에서는 스위치를 주로 사용

④ 마더보드 내장 이더넷 포트

  • 대부분의 마더보드에 기본 탑재되어 있으며, 주로 1Gbps를 지원 (일부 5Gbps / 10Gbps)
  • 제한된 내부 버퍼 크기와 낮은 성능으로 인해 고속 머신 비전 애플리케이션에는 권장하지 않음

3. USB 인터페이스

① USB PCIe 카드

  • PC의 USB 포트 수를 늘리는 PCIe 기반 확장 카드
  • USB 2.0, USB 3.0 등 다양한 표준과 USB-A, USB-B, USB-C, Micro-USB 커넥터를 지원하는 제품이 시판 중

② USB 포트 허브

  • 컴퓨터에 연결하여 USB 포트를 추가 확장하는 외부 장치
  • 추가 USB 장치 연결이 필요할 때 사용

③ 마더보드 내장 USB 포트

  • 요즘 대부분의 마더보드에는 USB 3.0 포트가 내장되어 있으며, 일부는 USB 2.0과 USB 3.0을 함께 지원

4. CoaXPress 인터페이스

① 프레임 그래버 (Frame Grabber)

  • 카메라에서 개별 프레임을 캡처하여 처리하는 전용 확장 카드
  • 주요 기능
    • 실시간 압축
    관심 영역(ROI) 오프로드로 추가 대역폭 활용
    다양한 고급 획득 및 이미지 처리 기능 지원

5. Camera Link / CLHS 인터페이스

① 프레임 그래버 (Frame Grabber)

  • Camera Link 카메라에서 프레임을 취득하는 전용 확장 카드
  • 주요 온보드 처리 기능
    • 프로그래밍 가능한 관심 영역(ROI)
    픽셀 데시메이션 (Pixel Decimation)
    이미지 스케일링 (Image Scaling)
    조회 테이블 (Lookup Table)
